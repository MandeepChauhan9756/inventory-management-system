# 📌 Overview

Inventory Pro is a full-stack Inventory Management System designed for managing products, customers and orders securely.

The application follows a modern client-server architecture where the frontend is built with React and communicates with a Flask REST API secured using JWT Authentication. Data is stored in PostgreSQL and the application can be deployed using Docker.

# 🚀 Installation

Clone Repository

```bash
git clone https://github.com/MandeepChauhan9756/inventory-management-system.git
```

Backend

```bash
cd backend

pip install -r requirements.txt

python app.py
```

Frontend

```bash
cd frontend

npm install

npm run dev
```

---

# 🐳 Docker

Run using Docker

```bash
docker-compose up --build
```
